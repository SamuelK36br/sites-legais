ADIVINHE A SENHA!

    Como jogar:
  1. Você precisa descobrir qual é a senha.
  2. Todas as senhas possuem 3 digitos.
    Menos o enigma bônus.
  3. A senha não possui números repitidos.
  4. As dicas te ajudarão descobrir qual é a senha.
  5. Boa sorte!
